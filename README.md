# student-grade-management-system

A beginner Python project that:
- Take student information as input
- Stores data using dictionaries
- Calculates total marks
- Calculate percentage
- Assigns grades based on percentage

  ## Concepts

  - Dictionaries
  - Nested Dictionaries
  - Input/Output
  - Conditional Statements
